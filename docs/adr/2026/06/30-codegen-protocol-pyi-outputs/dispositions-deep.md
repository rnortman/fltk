errhandling-1:
- Disposition: Fixed
- Action: `genparser.py` `gen_rust_cst` — replaced the `if protocol_output is not None and protocol_text is not None:` write guard with `if protocol_output is not None: assert protocol_text is not None; _write_output_file(...)`, mirroring the `init_pyi_text` writer below it (genparser.py:494-501).
- Severity assessment: The double-condition can never produce a diagnostic in current control flow (any generate_protocol error exits above), but it arms a silent-skip trap: a future error-recovery branch setting `protocol_text = None` would make the action exit 0 with `cst_protocol.py` absent and nothing logged, surfacing only as a downstream `ModuleNotFoundError`. The assert surfaces the invariant violation immediately.
